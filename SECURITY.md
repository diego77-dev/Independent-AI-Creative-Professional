# Security Policy

## Keeping Things Secure

Hey there! Security is important to me, even in a creative repository like this one. While this repo mainly contains creative prompts, workflows, and business resources rather than executables or code, I still take security seriously.

## Reporting a Vulnerability

If you discover a security issue in any of the content or resources here, please let me know by:

1. **Email**: Send details to [hello@shaunduval.com](mailto:hello@shaunduval.com) with "Security Issue" in the subject line
2. **Do not**: Create a public GitHub issue for security vulnerabilities

## What To Include In Your Report

When reporting a potential security problem, please include:

- A clear description of the issue
- Steps to reproduce it (if applicable)
- The potential impact as you see it
- Any suggestions for addressing the issue

## Security Considerations For This Repository

Since this is primarily a creative resource repository, security concerns might include:

- Problematic prompts that could generate harmful content
- Links to external resources that may have become compromised
- Personal information accidentally included in examples
- Potential misuse of AI techniques described here

## My Response Process

Here's what you can expect after reporting an issue:

1. **Initial response**: I'll acknowledge receipt of your report within 48 hours
2. **Assessment**: I'll evaluate the reported issue within 1 week
3. **Resolution**: If valid, I'll work to address the issue as quickly as possible
4. **Disclosure**: Once fixed, I'll update appropriate documentation and acknowledge your contribution (if you wish)

## Best Practices For Repository Users

If you're using resources from this repository:

- Always review AI prompts before using them with clients or in production
- Keep your AI tools and platforms updated
- Be mindful of how you apply creative techniques with sensitive content
- Consider the ethical implications of AI-generated content
- Verify external resources linked from this repository before use

## Thank You

Thanks for helping keep this creative resource safe and secure for everyone. By working together, we can make sure that AI creative tools and techniques are used responsibly and safely.
